
# SecureFile - Secure File Sharing Simulator

A simple, lightweight, and fully client-side secure file sharing simulator.

# Features
- Admin Login 
- Real-time AES-GCM Encryption (256-bit)
- PBKDF2 Key Derivation
- Share ID + Password based file sharing
- LocalStorage persistence
- No backend, no server required

# How to Run
1. Download `index.html`
2. Open the file in any browser (Chrome/Edge/Firefox)
3. To Login Contact Owner
   
# Cybersecurity Concepts Demonstrated
- Confidentiality (AES-GCM)
- Secure Key Derivation (PBKDF2)
- Integrity
- Client-side encryption

Tech Stack: Pure HTML + CSS + JavaScript + Web Crypto API
